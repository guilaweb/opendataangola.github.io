Nossa Missão

Promover a transparência governamental, a participação cidadã e o desenvolvimento socioeconômico por meio de dados abertos. Capacitar cidadãos, jornalistas e pesquisadores a usar dados para criar soluções inovadoras.

Nossos Parceiros

Direcção de Trânsito e Segurança Rodoviária

Colaborando na disponibilização de dados sobre segurança rodoviária.

Bolsa de Dívida e Valores de Angola

Parceiro na disponibilização de dados financeiros e de mercado.

Colaborando na promoção de dados abertos e transparência financeira.

Linha do Tempo

  • 2023: Fundação da DIANGUILA EMPREENDIMENTOS, LDA.

    Início das operações com foco em dados abertos e transparência.

    Realização de workshops sobre o uso de dados abertos.

  • 2024: Lançamento do primeiro conjunto de dados.

    Parcerias com instituições governamentais para disponibilização de dados.

    Desenvolvimento de ferramentas para visualização de dados.

    Participação em conferências sobre dados abertos.

  • 2025: Parcerias estratégicas para expandir o alcance.

    Colaboração com universidades e ONGs para capacitação em dados abertos.

    Promoção de hackathons para incentivar o uso de dados abertos.

    Desenvolvimento de uma plataforma online para acesso a dados.

    Participação em eventos internacionais sobre dados abertos.

    Promoção de iniciativas de cidadania digital.
